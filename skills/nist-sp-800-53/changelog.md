# Changelog — NIST SP 800-53 Skill

All notable changes to this skill are documented here. Format loosely follows
[Keep a Changelog](https://keepachangelog.com/). Framework-content changes should
land via pull request after verification against the canonical sources.

## [Unreleased]

### Added
- **Structured reference content.** Bundled the full control catalog under
  `references/` (index + 20 per-family files, ~1,196 controls incl.
  enhancements), generated by `references/ingest.py` from the official OSCAL
  catalog (`usnistgov/oscal-content`, public domain). Added a `## Reference Data`
  section to SKILL.md instructing Claude to cite the bundled content as
  `[OFFICIAL]` via progressive disclosure.
- `content_mode: structured` and `reference_index` in framework_profile.yaml.

### Changed
- Pinned version `5.1.0` -> `5.2.0` to match the current OSCAL catalog
  (`metadata.version` 5.2.0). Verify `publication_date` against the canonical
  source.

### Added (initial scaffold)
- Initial skill scaffold: SKILL.md, framework_profile.yaml, sources.yaml,
  source_state.json, tests.
- Source monitoring entries for SP 800-53 Rev. 5, 800-53A, 800-53B, the controls
  project hub, and the OSCAL content repository.
